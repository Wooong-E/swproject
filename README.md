# 경산시 관광 웹 서비스 프로젝트

## 프로젝트 개요
경산시 관광 관련 정보를 제공하는 웹 서비스입니다.  
사용자가 관광지를 검색하고 세부 정보를 확인할 수 있도록 구현되었습니다.  
이 프로젝트는 SW대회 참가용으로 진행 중입니다.

---

## 팀 구성
| 역할 | 이름 | 담당 |
|------|------|------|
| 개발 | 조웅 | 웹 프론트엔드 구현, Git repository 관리 |
| 개발 | 서상범 | 백엔드 기능 개발 |
| 개발 | 이택민 | DB 연동 |

> 디자인 및 기획 관련 기능은 시각디자인과, 경영학과 팀원이 담당합니다.

---

## 주요 기능
- **관광지 검색 및 필터링**: 카테고리별 관광지 조회  
- **관광지 상세 정보 확인**: 사진, 설명, 위치 등  
- **관리자 기능**: 관광지 등록/수정, 사용자 관리, 통계 확인 (예정)  

---

## 기술 스택
- **프론트엔드**: HTML, CSS, JavaScript 
- **백엔드**: Java, Spring Framework  
- **데이터베이스**: MySQL  
- **버전 관리**: Git & GitHub
